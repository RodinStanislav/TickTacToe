ConsoleColor - хранить энам с цветами для символов.

Point2 - структура (x, y). Аналог POINT (из Windows.h)

GameParams - Все основные переменные игры. Манипуляция с ними позволит изменить "правила"
Например, можно задать цвета для символов, количество символов на доске (стандартно 3x3).
Можно изменить размер символов, также задать разрешение консоли и отступ по x, y

ConsoleManipulator - Все основные действия с консолью. Работа с кареткой. 

InfoPrinter - "Меню" игры. Предполагался больший функционал, вроде отображения счета или текущего символа

InputHandler - Ловит события мыши. (По хорошему, переименовать бы в файл MouseInputHandler, так как за 
клавиатуру отвечает InfoPrinter)

GameMap - Работа с игровом полем. Обработка установки символа, определение выигрыша

Player AI - ИИ соперник

main - собирает все в кучу и делает это работоспособным
